const boost::dispatch::functor<tag::atan2d_> boost::simd::atan2d = {}

Function object implementing atan2d capabilities.

atan2d function : atan2 in degrees.

Semantic:

For every parameters of floating type T:

T r = atan2d(x, y);

is similar but not fully equivalent to:

T r = atand(y/x);;

as it is quadrant aware.

For any real arguments x and y not both equal to zero, atan2d(x, y) is the angle in degrees between the positive x-axis of a plane and the point given by the coordinates (y, x).

It is also the angle in \([-180,180[\) for which \(x/\sqrt{x^2+y^2}\) and \(y/\sqrt{x^2+y^2}\) are respectively the sine and the cosine.

template<typename T >
auto boost::simd::toints ( T const &  x)

Convert to integer by saturated truncation.

semantic:
For any given value x of type T:
as_integer_t<T> r = toints(x);

The code is similar to:

as_integer_t<T> r = static_cast<as_integer_t<T> >(saturate<as_integer_t<T> >(x))
Notes:
  • The Inf, Minf and Nan values are treated properly and go respectively to Valmax, Valmin and Zero of the destination integral type.
  • All values superior (resp.) less than Valmax (resp. Valmin) of the return type are saturated accordingly.
  • If you do not care about invalid values or overflows, toint is faster.
Alias

ifix, itrunc

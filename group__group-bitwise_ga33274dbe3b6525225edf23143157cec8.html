template<typename T , template N>
auto boost::simd::rshl ( T const &  x,

Returns the first entry shifted left or right by the absolute value of the second entry, according to its sign.

semantic:
For any given value x of type T, n of type I:
T r = rshl(x, n);
See also
rshr, shr, shl
Returns
a value of the same type as the first input.

template<typename T>
struct boost::simd::cardinal_of< T >

Computes the register width required to store an instance of any type T.

Models:

Integral Constant

Semantic:

For any given cv-qualified type T,

using r = cardinal_of<T>::type;

is equivalent to:

using r = sd::integral_constant<std::size_t,sizeof(T)/sizeof(boost::dispatch::scalar_of_t<T>)>;
Template Parameters
TType which register size is to be computed.

Computes the (saturated) absolute value of the difference of its parameters.

semantic:
For any given value x, y of type T:
T r = dists(x, y);

is similar to:

T r = abss(x-y);
Note

The result is never stricly negative moreover it satisfies the predicate

is_positive(dist(x, y)) || is_unord(x, y)

For floating, it can of course be Nan.

See also
abss, ulpdist
Returns
a value of the same type as the input.

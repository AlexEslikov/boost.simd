const boost::dispatch::functor<tag::rem_pio2_> boost::simd::rem_pio2 = {}

Function object implementing rem_pio2 capabilities.

Computes the remainder modulo \(\pi/2\).

Semantic:

For every parameter of floating type T

T r, rc;
as_integer<T> n;
n = rem_pio2<Range>(x, r);

is similar to:

as_integer<T> n = div(iround2even, x, Pio_2<T>());
T r = remainder(x, Pio_2<T>());
Note:

The Range parameter is a type among big_, medium_, small_, very_small_ that allow to statically choose the computation process.

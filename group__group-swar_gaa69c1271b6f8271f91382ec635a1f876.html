template<typename T >
auto boost::simd::enumerate ( T const &  x)

Linear enumeration of value.

Return a SIMD register containing a linear enumeration of value defined by a seed value and a step.

Semantic:

For any given SIMD type T, and any value seed and step of a scalar type S, the following code :

T r = enumerate<T>(seed, step);

is equivalent to

T r = make<T>(seed, seed + step, ... , seed + (N-1)*step);

where N is the equal to cardinal_of<T>::value.

For any given SIMD type T, and any value seed and step of a SIMD type S, the following code :

T r = enumerate<T>(seed, step);

is equivalent to

T r = splat<T>(seed) + splat<T>(step)*enumerate<T>();
Returns
A SIMD register of scalar type T

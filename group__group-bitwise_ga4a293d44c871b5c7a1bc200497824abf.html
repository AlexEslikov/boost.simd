template<typename T0 , typename T1 >
auto boost::simd::bitwise_xor ( T0 const &  x,
T1 const &  y 
)

return the bitwise xor of the two parameters

The operands must share the same bit size.

The result type is the one of the first operand.

Infix notation can be used with operator '^', but this will not work in scalar mode if any operand is floating point because of C++ limitations.

Semantic:

For every parameters of types respectively T0, T1:

T0 r = bitwise_xor(x,y);

is similar to:

T0 r = x^y;
Alias:
b_xor
See also
bitwise_or, bitwise_and, bitwise_notand, bitwise_andnot, bitwise_notor, bitwise_ornot, complement
Returns
a value of the same type as the first parameter

const boost::dispatch::functor<tag::is_greater_equal_> boost::simd::is_greater_equal = {}

Function object implementing is_greater_equal capabilities.

Returns True or False according x is greater or equal to y or not.

Infix notation can be used with operator '>='.

Semantic:

For every parameters of types respectively T0, T1:

as_logical_t<T0> r = is_greater_equal(x,y);

is similar to:

as_logical_t<T0> r = x >= y;
Alias:
ge, is_ge

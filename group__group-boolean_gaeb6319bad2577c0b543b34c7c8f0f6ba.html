template<typename T1 , typename T2 >
auto boost::simd::logical_and ( T1 const &  x,
T2 const &  y 
)

return the logical and of the two parameters.

The operands must share the same cardinal_of value.

Infix notation can be used with operator '&&'.

Semantic:

For every parameters of types respectively T0, T1:

as_logical_t<T0> r = logical_and(x,y);

is similar to:

as_logical_t<T0> r = x && y ;
Alias:
l_and
See also
logical_or, logical_xor, logical_notand, logical_andnot, logical_notor, logical_ornot, logical_not
Returns
a value of the logical type associated to the first parameter

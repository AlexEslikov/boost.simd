template<typename T, std::size_t N>
class boost::simd::pack< T, N >

High-level interface for manipulating SIMD data.

For a given couple T and N, provides an interface to manipulate SIMD data with the best storage possible depending on your hardware.

Precondition
N must be a power of two.
Template Parameters
TType of the stored elements
NNumber of element stored

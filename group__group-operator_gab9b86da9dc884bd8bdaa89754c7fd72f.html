const boost::dispatch::functor<tag::logical_not_> boost::simd::logical_not = {}

Function object implementing logical_not capabilities.

return the logical negation of the parameter, i.e. True or False of the logical type associated entry type according that the input is Zero or non Zero (False or True).

Infix notation can be used with operator '!'

Semantic:

For every parameter of type T0

as_logical_t<T0> r = logical_not(x);

is similar to:

as_logical_t<T0> r =!x;
Alias:
l_not

template<typename T >
auto boost::simd::signgam ( T const &  x)

sign of gamma function

Semantic:

For every parameter of floating type T0

T0 r = signgam(x);

is similar to

T0 r = sign(gamma(x));

signgam does not computes gamma and is fast.

See also
gamma, gammaln
Returns
a value of the same type as the parameter

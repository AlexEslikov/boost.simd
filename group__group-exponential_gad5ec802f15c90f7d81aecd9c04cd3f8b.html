const boost::dispatch::functor<tag::nthroot_> boost::simd::nthroot = {}

Function object implementing nthroot capabilities.

nth root function: \(\sqrt[n]{x}\)

  • n must be of integer type
  • if n is even and x negative the result is Nan
  • if x is null the result is Zero
  • if x is one the result is One
Semantic:

For every parameters of floating type T and integral type N:

T r = nthroot(x, n);

is similar to:

T r = n >= 0 ? pow(x, rec(tofloat(n))) : Nan;
Note:
nthroot is more expansive than pow(x, rec(tofloat(n))) because it takes care of some limits issues that pow does not mind of.

auto boost::simd::rem ( T const &  x,
T const &  y 
)

Computes the remainder of division.

The return value is x-n*y, where n is the value x/y, rounded toward zero.

semantic:
For any given value x, y of type T:
T r = rem(x, y);

For floating point values the code is similar to:

T r = x-divfix(x, y)*y;

For floating entries:

  • if x is +/-inf , Nan is returned
  • if x is +/-0 and y is not 0 x is returned
  • If y is +/-0, Nan is returned
  • If either argument is NaN, Nan is returned

If correct values for these limit cases do not matter for you, calling remainder(x, y, fast_) can gain some cycles.

The returned value has the same sign as x and is less than y in magnitude.

A call to rem(x, std_) uses the std::libc++ std::fmod function

Alias

fmod, remfix

See also
remainder, mod, modulo
Returns
a value of the same type as the input.

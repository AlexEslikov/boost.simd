template<typename T >
auto boost::simd::arg ( T const &  x)

Computes the angular orientation of its parameter.

semantic:
For any given value x of floating type T:
T r = arg(x);

is equivalent to:

T r = (is_nan(x)) ? x : (is_negative(x)) ? Pi<T>() : Zero<T>();
Note:
  • Returns \(+0\) or \(\pi\) (or Nan if and only if the input is Nan) according to the input sign bit,
  • arg(x, use_signbit_) always returns \(+0\) or \(\pi\) taking into account the bit of sign even in the Nan case and is a bit speedier.
  • This function is the restriction to real numbers of the complex arg(ument) function.
Alias:

angle

Ieee functions
User API » Functions and Constants

Description

Those functions provides scalar and SIMD algorithms for inspecting, generating or decomposing IEEE 754 floating point numbers.

Operations like exponent and mantissa extraction, floating point modulo, IEEE bit patterns manipulation and magnitude comparison are provided.

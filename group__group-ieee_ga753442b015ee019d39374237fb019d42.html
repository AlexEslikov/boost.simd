template<typename T , typename N >
auto boost::simd::predecessor ( T const &  x,
N const &  n 
)

Returns the n-th greatest element strictly less than the parameter.

Semantic:
T r = predecessor(x,n);

computes the n-th greatest representable value strictly less than x in its type. n must be positive or null.

For integer it saturate at Valmin, for floating point numbers, all Minf strict predecessors are Nan.

See also
next, prev, successor, nextafter, Minf, Valmin, Nan

a value of same type as the inputs

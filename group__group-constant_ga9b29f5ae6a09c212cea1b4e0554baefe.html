auto boost::simd::Constant ( )
noexcept

Generates a value of the chosen type from an exact bits pattern.

Constant generates a value from an exact bits pattern that get splat in the return value.

Semantic:

or any type T and any integral constant N :

T r = Constant<T,N>();

generates a values which bit pattern can be represent by the integer N.

Template Parameters
TypeType of the desired value
BitsBits pattern to generate as a value
Returns
A value of type T which bits are exactly equals to N

const boost::dispatch::functor<tag::divfloor_> boost::simd::divfloor = {}

Function object implementing divfloor capabilities.

Computes the floor of the division.

semantic:
For any given value x, y of type T:
T r = divfloor(x, y);

The code is similar to:

T r = floor(x/y);

for integral types, if y is Zero, it returns Valmax or Valmin if x is positive (resp. negative) and Zero if x is Zero. Take also care that dividing Valmin by -1 for signed integral types has undefined behaviour.
